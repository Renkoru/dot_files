Ansible plays to configure VPN

* Setup SSH on VPS
#+NAME: copy SSH key
#+BEGIN_SRC bash
  ssh-copy-id <USER>@<IP_ADDR>
#+END_SRC

* Simple commands
#+BEGIN_SRC bash
  poetry run ansible all -m ping -i inventory.ini
  poetry run ansible-playbook -i inventory.ini --ask-vault-password --ask-become-pass vpn_setup.yaml
#+END_SRC
